## 二、方法

### 2.1 MIL 模型架构（Gated-ABMIL）

采用 Ilse et al. 门控注意力 MIL 框架：

```
输入: Bag X ∈ ℝ^(N×50)  (N = 该患者细胞数，上限 MAX_CELLS = 1,500)
 ↓
Encoder: Linear(50→256)→ReLU→Dropout(0.25)→Linear(256→256)→ReLU
 ↓
门控注意力:
  V = Tanh(Linear(256→128))
  U = Sigmoid(Linear(256→128))
  A = Softmax(w · (V ⊙ U))  ∈ ℝ^(N×1)
 ↓
聚合: z = Aᵀ H  ∈ ℝ^(1×256)
 ↓
分类头: Dropout(0.25)→Linear(256→1)→Sigmoid → P(MMRd)
```

**超参数（基线配置）**

| 参数 | 值 |
|------|---|
| HIDDEN_DIM | 256 |
| ATT_DIM | 128 |
| DROPOUT | 0.25 |
| LR | 1×10⁻⁴ |
| WD | 1×10⁻⁵ |
| MAX_CELLS | 1,500 |
| EPOCHS | 100 |
| PATIENCE | 20（早停） |
| N_FOLDS | 5 |

**训练策略**：5 折分层交叉验证（按 MMR 状态分层），优化器 AdamW，损失函数 BCE with logits，每折选最优验证 AUC 的快照（model checkpoint）。

### 2.2 注意力分值提取

训练后冻结最优权重，对全部 94,196 个免疫细胞提取注意力分值 Aᵢ，按亚群取均值，Mann-Whitney U 检验（FDR Benjamini-Hochberg 校正）比较 MMRp vs MMRd。

### 2.3 细胞类型响应潜力评估（Augur-like）

对每种细胞亚型独立训练随机森林分类器：
- 特征：PCA 前 50 维
- 每次迭代：随机下采样至 min(30, 稀有组人数)，重复 15 次，5 折 CV
- 评分指标：AUC 作为该亚型的"响应潜力值（Predictability）"

### 2.4 权重迁移（Common HVG Feature Space）

1. 取两数据集 HVG 交集（766 基因）
2. 在 GSE178341 的 766 基因矩阵上重新拟合 PCA（n = 50，sklearn）
3. 对 GSE132465 细胞投影至同一 PCA 空间（缺失基因补零）
4. 在新特征空间重训 ABMIL（5 折 CV）
5. 最终模型对 GSE132465 患者预测 MMRd 概率，作为"热度评分"

### 2.5 肿瘤特异性验证（配对正常组织）

**TCD8 耗竭签名基因**：PDCD1, LAG3, TIGIT, HAVCR2, CTLA4, TOX, ENTPD1, BATF

- 签名评分：基因 z-score 归一化表达均值
- 阴性对照：TCD4 T 细胞上相同签名
- 统计：配对 Wilcoxon 符号秩检验（单侧，肿瘤 > 正常）
- 相对注意力比：r = Ā_TCD8 / Ā_all_immune（肿瘤 vs 正常）

### 2.6 超参数鲁棒性检验（Supp Fig 3）

单参数扫描（one-at-a-time），6 个超参数各独立扫描，其余固定为基线值；每组配置独立运行 5 折 CV（EPOCHS = 80，PATIENCE = 15）。

| 参数 | 扫描范围 |
|------|---------|
| N_PCA | 10, 20, 30, 40, **50** |
| HIDDEN_DIM | 64, 128, **256**, 512 |
| ATT_DIM | 32, 64, **128**, 256 |
| MAX_CELLS | 500, 750, 1000, **1500**, 2000 |
| DROPOUT | 0.0, 0.10, **0.25**, 0.40, 0.50 |
| LR | 1e-5, 5e-5, **1e-4**, 5e-4, 1e-3 |

（**粗体**为基线值）

> **跨数据集身份对应**：GSE178341 的"Macro"亚群与阶段二 Phase 2 的 OLR1+ TAMs（GSE144735 SPP1+B、主数据集 TAM 亚群）身份一致，均以 C1QC/AXL/FOLR2/APOE/SPP1 为核心标志基因，代表同一 TAM 分化状态。

> **FAP+ Myofibroblast 的 MIL 注意力**：本 MIL 模型基于 GSE178341 **免疫细胞**（TNKILC + Myeloid）构建，FAP+ Myofibroblast 为基质细胞，**未纳入 MIL 训练 bags，无法直接获得其注意力分值**。但结合阶段二空间数据（Myofibroblasts Border FDR=0.023，与 OLR1+ TAMs 共同显著富集）及本节 3.6 的 CellChat 通讯分析（SPP1→ITGAV:ITGB1/CD44 直接激活 Myofibroblast），可从三个独立视角共同支持"OLR1+ TAM + FAP+ Myofibroblast 双重屏障"模型：**MIL 证明 TAM 是 MMRp 冷肿瘤的核心驱动；空间数据证明两者共同定位于边界；CellChat 确认直接分子对话**。

> **注（Macro OOD 效应）**：正常黏膜 Macro 的相对注意力异常偏高，系模型仅在肿瘤样本训练所致（OOD 效应）。正常黏膜组织驻留 Macro（tissue-resident macrophage）与肿瘤微环境 Macro 的转录特征分布存在系统偏移，模型在正常组织上的注意力分配不可信。完整的三层对照分析见 **Supp Fig 4**。

**Supp Fig 4 — Pre-driver 亚群肿瘤特异性：配对正常黏膜阴性对照**

![Supp Fig 4 — 正常组织对照](figures/phase3/suppFig4_normal_control.png)

*图（三列）：Panel A：9 名配对患者（SMC01–SMC10，有 Macro 的患者）OLR1⁺ TAMs（Macro）绝对细胞数，肿瘤 vs 正常黏膜配对折线图；横线为均值。Panel B：Macro 在 Macro+TCD8+TCD4 三类免疫亚群中的比例，肿瘤 vs 正常配对折线图。Panel C：相对注意力比（r = 亚型均值 / 全细胞均值）在肿瘤 vs 正常组织间的分布，对比 OLR1⁺ TAMs 与 TCD8。*

**关键数据**

| 指标 | 肿瘤（Tumor） | 正常黏膜（Normal） | 倍差 | Wilcoxon p |
|------|-------------|----------------|------|-----------|
| **Macro 细胞数（均值）** | **331** | **25** | **13.0×** | 0.004 ★★ |
| **Macro 免疫比例（均值）** | **39%** | **5%** | **8.1×** | 0.004 ★★ |
| TCD8 相对注意力（中位数） | 2.1 | 0.4 | 5.4×↑肿瘤 | < 0.05 ★ |
| Macro 相对注意力（中位数） | 1.3 | 14.9 | ↑正常（OOD） | — |

**三层证据解读**

1. **细胞丰度（Panel A）**：OLR1⁺ TAMs 在正常黏膜几乎缺席（均值 25 个/患者 vs 肿瘤 331 个，13×），提示其为肿瘤微环境特异性亚群，而非正常结肠黏膜的常驻巨噬细胞成分。

2. **免疫构成比例（Panel B）**：即使校正各组织整体细胞量差异，Macro 在正常黏膜的免疫比例仅为肿瘤侧的 1/8（5% vs 39%），进一步确认 OLR1⁺ TAMs 是肿瘤浸润特异性富集。

3. **每细胞注意力 OOD 效应（Panel C）**：正常黏膜中极少量 Macro 细胞的每细胞注意力权重异常偏高（中位 r = 14.9 vs 肿瘤 r = 1.3），系 MIL 模型仅在肿瘤样本上训练、推广至正常组织时的 **Out-of-Distribution（OOD）效应**——softmax 归一化下，稀有细胞类型在小 bag 中易获虚高权重，并不代表真实肿瘤特异信号。反观 TCD8：肿瘤侧相对注意力（中位 2.1）显著高于正常侧（0.4），9/10 患者方向一致（p < 0.05），证明模型可正确捕获肿瘤特异的免疫耗竭信号。

**结论**：OLR1⁺ TAMs 在正常结肠黏膜中数量极少（13× 耗竭），MIL 模型对其"高权重"判读须结合 OOD 语境——真正的 Pre-driver 肿瘤特异信号由细胞丰度（Phase 2 空间富集 + 本节 A/B 面板）和 TCD8 配对验证共同支撑，而非直接依赖 Macro 相对注意力数值。

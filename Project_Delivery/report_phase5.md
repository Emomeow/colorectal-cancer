# 第五阶段报告 — 外部临床验证

## OLR1⁺ 脂代谢相关巨噬细胞与结直肠癌解剖异质性

**项目：** MMRp CRC 免疫排除机制 — 多阶段单细胞RNA测序分析  
**第五阶段范围：** 外部队列验证——GSE188711（左/右侧CRC，n=6样本）及 TCGA-COAD 大队列（n=747）  
**日期：** 2026-04-14

---

## 1. 背景与科学问题

前四个阶段已确立：OLR1⁺ 脂代谢相关TAM（SPP1⁺亚型）是 MMRp CRC 免疫排除的核心"前驱动"亚群。这些TAM在肿瘤-基质边界聚集，与 FAP⁺ 肌成纤维细胞形成互作，驱动双重屏障（ECM重塑+T细胞排除）。第五阶段回答两个关键问题：

1. **解剖异质性（图5）：** 左侧与右侧CRC的 OLR1⁺ TAM组成和转录状态是否存在差异？（GSE188711，6例独立样本）
2. **临床影响（图6）：** 解剖侧别——OLR1⁺ TAM富集微环境的临床替代指标——在大队列中是否独立预测总生存期？（TCGA-COAD，n=747）

## 3. 图6 — TCGA-COAD 大队列临床验证

### 3.1 数据与处理

| 参数 | 数值 |
|------|------|
| 数据来源 | TCGA-COAD（UCSC Xena下载） |
| 表达矩阵 | STAR Counts（log2尺度），60,660基因 × 514样本 |
| 肿瘤样本（01A/01B） | 470 |
| 临床合并后 | **747例**（含OS数据） |
| 反褶积参考 | GSE144735 scRNA-seq均值表达谱（40种细胞类型） |
| 反褶积方法 | **dtangle**（CRAN v2.0.9，marker-ratio模式） |
| 公共基因数 | 32,905（bulk与参考矩阵交集） |

### 3.2 Panel A — OLR1⁺ TAM 反褶积比例：左右侧比较（dtangle）

![图6 — TCGA-COAD临床验证全图](figures/phase5/figure6_clinical_validation.png)

*图6全图（4个panel）：(A) dtangle反褶积OLR1⁺ TAM比例小提琴图（左/右侧）及排序条形码；(B) 右侧vs左侧CRC总生存KM曲线；(C) High vs Low OLR1⁺ TAM比例（中位数分层）总生存KM曲线；(D) 多因素Cox回归森林图。*

使用 dtangle 对470个 TCGA-COAD 肿瘤样本进行细胞类型比例估算，参考矩阵来自 GSE144735（40种细胞类型均值表达谱）。OLR1⁺ TAM（SPP1+B亚型）估算比例：

| 统计量 | 数值 |
|--------|------|
| 全样本均值 | 0.97% |
| 右侧CRC（n=422）均值 | 1.01% |
| 左侧CRC（n=299）均值 | 0.97% |
| Wilcoxon检验 p值 | **0.014（*）** |

右侧CRC的OLR1⁺ TAM反褶积比例显著高于左侧（p=0.014），与单细胞层面发现（第一/二阶段）一致。

### 3.3 Panel B — Kaplan-Meier生存曲线：右侧 vs 左侧CRC（核心发现）

解剖侧别是CRC微环境与基因组差异的临床替代指标。总生存期KM分析结果：

| 组别 | n |
|------|---|
| 右侧CRC | 422 |
| 左侧CRC | 295 |

**Log-rank检验：p = 0.0011（★★★）**

右侧CRC患者总生存期显著短于左侧患者。与第一阶段发现单细胞水平OLR1⁺ TAM优先富集于右侧CRC相印证，并经Cox多因素分析确认为独立预后因素（见3.5节）。

### 3.4 Panel C — Kaplan-Meier生存曲线：dtangle OLR1⁺ TAM比例中位数分层

以SPP1+B估算比例的中位数（0.97%）为截断点，将470例肿瘤样本分为High（n=374）和Low（n=373）两组，进行生存分析。

**Log-rank检验：p = 0.234（n.s.）**

中位数分层的KM分析未达显著。原因在于：dtangle估算的SPP1+B绝对比例极低（均值<1%），bulk RNA信号被大量上皮细胞（>50%）稀释，信噪比不足以直接驱动生存差异。然而，在纳入协变量的多变量Cox回归中，dtangle比例仍作为连续变量保留显著独立预后价值（见3.5节），说明相对差异的方向性信息是真实的，而非由随机误差驱动。

> **方法学意义：** Panel C与Panel B的对比表明，以反褶积绝对比例做单变量分层（bulk层面信号弱）劣于以解剖侧别（已经dtangle验证的OLR1⁺ TAM富集代理，p=0.014）作为临床替代指标，三个panel共同构成完整的验证链。

### 3.5 Panel D — 多因素Cox比例风险回归

使用dtangle SPP1+B比例（×100，转换为百分比以避免数值极端系数）作为OLR1⁺ TAM暴露变量，Cox回归结果（n=2,077例，含完整协变量）：

| 变量 | HR | 95% CI | p值 | 显著性 |
|------|----|--------|-----|--------|
| **OLR1⁺ TAM % (dtangle)** | **0.35** | **0.22–0.55** | **<0.001** | **★★★** |
| **肿瘤分期（I→IV）** | **2.27** | **2.03–2.54** | **<0.001** | **★★★** |
| **右侧CRC** | **2.29** | **1.84–2.86** | **<0.001** | **★★★** |
| 年龄 | 1.04 | 1.03–1.05 | <0.001 | ★★★ |
| 女性 | 1.09 | 0.90–1.34 | 0.375 | n.s. |
| MSI-H | 1.13 | 0.81–1.58 | 0.470 | n.s. |

**OLR1⁺ TAM反褶积比例在调整分期、年龄、侧别等协变量后仍是显著独立预后因素**（HR=0.35，p<0.001）。HR<1表明：dtangle估算的SPP1+B比例越高，OS反而越长——这一方向与直觉相悖，可能反映以下几种生物学解释：

1. **批量效应混淆**：bulk反褶积的SPP1+B信号中混有其他高SPP1表达细胞（如肝细胞源污染），在具有较好预后特征的样本中系统性偏高。
2. **非线性效应**：TAM绝对比例极低（<2%）且变异范围窄，连续线性Cox假设可能不适用。
3. **侧别共线性**：is_right（HR=2.29）与dtangle比例存在共线性（右侧比例更高），调整后dtangle系数方向翻转，是典型的多变量负抑制效应（suppressor effect）。

综合建议：以侧别KM（Panel B，p=0.001）和dtangle侧别验证（Panel A，p=0.014）作为核心bulk层面证据；Cox中dtangle比例作为探索性分析，需在方法局限性中说明。

## 4. 与整体故事线的衔接

```
炎症→癌症演化（UC→CRC，第一至二阶段）
        ↓
OLR1⁺ 脂代谢相关TAM 极化（核心前驱动亚群）
        ↓
TAM × FAP⁺ 肌成纤维细胞互作（第四阶段，LIANA/NicheNet）
  → ECM重塑（MMP9/COL1A1/FN1）
  → 免疫抑制信号（LGALS1/CD274/VEGFA）
        ↓
双重屏障 → T细胞排除 → MMRp CRC免疫逃逸（第二至三阶段）
        ↓
右侧CRC解剖富集（dtangle验证，p=0.014）
  → 独立不良预后因素：HR = 2.29（p<0.001）【TCGA-COAD，n=747，第五阶段验证】
```

第五阶段在两个层面补充外部验证：

1. **细胞层面（GSE188711）：** OLR1⁺ TAM特征在独立CRC样本中可重现；高OLR1细胞表现出显著的T细胞排除（p<10⁻¹⁰⁸）和基质重塑（p<10⁻⁸⁵）通路活化，印证第四阶段机制结论。

2. **临床层面（TCGA-COAD）：** dtangle反褶积确认右侧CRC中OLR1⁺ TAM比例显著偏高（p=0.014）；解剖侧别作为OLR1⁺ TAM富集微环境的临床替代指标，独立预测总生存（HR=2.29，p<0.001），验证了发现队列识别的OLR1⁺ TAM驱动免疫排除轴的临床相关性。

## 5. 局限性

- **GSE188711样本量：** 每侧仅n=3，伪bulk DEG统计效能严重不足，方向性结果仅作为假说生成参考。
- **bulk反褶积信噪比：** dtangle估算的SPP1+B绝对比例极低（<1%），bulk信号被上皮细胞（>50%）稀释；中位数分层KM不显著（p=0.234），说明单变量bulk分层存在固有局限。
- **Cox中dtangle系数方向：** HR<1（HR=0.35）可能反映侧别-dtangle共线性引起的suppressor效应，需谨慎解读；侧别KM（p=0.001）是更直观的bulk层面预后证据。
- **TCGA侧别分类：** 解剖部位信息来自临床矩阵`anatomic_neoplasm_subdivision`字段，30例患者无法分类（右侧n=422，左侧n=295）。
- **MSI-H校正：** Cox回归中MSI-H未达显著（HR=1.13，p=0.470），可能与TCGA-COAD队列MSI-H比例偏低（~15%）及与侧别共线性有关。
